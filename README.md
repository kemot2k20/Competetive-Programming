# Competitive Programming Library

A comprehensive, categorized collection of optimized algorithms and data structures implemented in **C++**. 

This repository serves as my personal reference library, developed and refined during my preparations for the **Polish Olympiad in Informatics (OI)**, **AMPPZ** (Polish Collegiate Programming Contest), and various online competitive programming platforms.

## 🗂️ Repository Structure

The algorithms are categorized by their primary domain. Each folder contains clean, tested, and highly optimized implementations ready to be used in competitive programming environments.

## 🛠️ Tech Stack & Compilation

* **Language:** C++ (Standard C++17 / C++20)
* **Focus:** Time complexity optimization ($\mathcal{O}$ analysis), low memory footprint, and constant-factor optimization.

Most files are standalone and can be compiled using standard `g++` flags typically used in competitive programming:

```bash
g++ -O2 -std=c++17 -Wall -Wextra -Wshadow algorithm_name.cpp
